# 🎬 IMDB Vintage Movies Ratings Analysis

A data analysis project focused on discovering what makes a movie successful on IMDB — exploring genres, directors, writers, runtime, and audience engagement.

---

## 📘 2. Introduction

### 🎯 Purpose  
To understand the common patterns and success metrics behind top-rated movies on IMDB using data-driven analysis.

### ✅ Objective of the Project  
The aim is to evaluate IMDB's highest-rated movies by examining variables such as genre, director, writer, and runtime, and how these influence rank, votes, and average ratings.

### 🧩 Problem Being Addressed  
This project explores what factors contribute to a film's success. The analysis aims to answer: *What patterns exist in top-rated movies, and how can the film industry leverage them?*

### 📊 Key Datasets and Methodologies  
- **Dataset**: Publicly available IMDB Top Movies dataset  
- **Tools**: Microsoft Excel  
- **Techniques**: Pivot tables, `VLOOKUP`, `COUNTIF`, conditional formatting, filtering, charting

---

## 📖 3. Story of Data

### 📥 Data Source  
- IMDB public dataset via Kaggle.

### 🧾 Data Collection Process  
- Exported as CSV; downloaded and cleaned in Excel.

### 🗂️ Data Structure  
- **Rows**: Each row = one movie  
- **Columns**: Title, genre, director, writer, rating, runtime, votes, rank, year

### 🔑 Important Features  
- **Primary Title, Genre, Directors, Writers** – for qualitative insights  
- **Rank, Rating, Votes, Runtime** – for performance evaluation  

### ⚠️ Data Limitations  
- Some incomplete entries  
- Bias toward English/global films  
- Ratings based on public votes, subject to skew

## 🔍 5. Pre-Analysis

### 📊 Key Trends  
- Most common genres: Comedy, Drama, Romance  
- Peak year: 2014

### 🔗 Early Correlations  
- Long runtimes linked to higher ratings  
- Certain directors consistently produced high-ranking films

### 🕵️ Initial Insights  
- Christopher Nolan & Quentin Tarantino stood out  
- High vote count often aligned with high ratings

---

## 📈 6. In-Analysis

### 📌 Unconfirmed Insights  
- Director impact stronger than genre in some cases  
- Comedy gets high engagement but lower average ratings

### 💡 Recommendations  
- Invest in top-tier director/writer combos  
- Promote underrated genres  
- Replicate success from peak years like 2014

### 🛠️ Excel Techniques Used  
- Pivot Tables    
- Conditional Formatting  
- Bar/Line/Scatter Charts

---

## 📌 7. Post-Analysis and Insights

### 🏆 Key Findings  
- **Top Movie**: *The Shawshank Redemption*  
- **Top Genres**: Comedy, Drama, Romance  
- **Best Director**: Christopher Nolan  
- **Top Writer**: Quentin Tarantino  
- **Breakout Year**: 2014

### 🧠 Compare to Initial Assumptions  
- Genre expected to dominate, but directors had stronger influence

---

## ✅ 8. Recommendations & Observations

### 📌 Actionable Insights  
- Invest in writer/director talent  
- Boost low-ranking movies with good genre but poor exposure  
- Focus marketing on high-ROI genres and years

### 🔄 Business Strategy  
- Leverage historical success patterns  
- Promote genre diversity through data-backed decisions

### ⚡ Unexpected Discoveries  
- High-rated niche films had fewer votes  
- Horror and sci-fi underrepresented despite loyal fans

---

## 🧾 9. Conclusion

### 📚 Learnings  
- Success isn't genre-bound — director and writer power matters  
- 2014 was an exemplary year for quality productions  
- Excel is a powerful tool for movie analytics!

### ⚠️ Limitations  
- Dataset may exclude indie/international content  
- Viewer bias in ratings  
- Lack of financial (box office) data

### 🔍 Future Research  
- Integrate box office earnings  
- Predictive modeling using viewer demographics  
- Compare against streaming performance
